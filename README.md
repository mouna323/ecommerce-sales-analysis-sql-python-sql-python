# Customer Sales Analysis using SQL & Python

##  Project Overview
This project analyzes an e-commerce dataset using SQL and Python to extract business insights.
The goal is to understand sales performance, customer behavior, and profitability.

##  Key Questions Answered
- What is the total revenue and profit?
- Which regions generate the most sales?
- Who are the top customers?
- How does revenue change over time?
- Do high sales always mean high profit?

##  Methodology
1. Data cleaning using Python (Pandas)
2. Data loading into SQLite database
3. SQL queries for business analysis
4. Visualization using Matplotlib & Seaborn
5. Insight generation

## Key Insights
- APAC region generates the highest sales → focus marketing there  
- Sales vary over time → plan promotions in low periods  
- A few customers generate most revenue → retain them  
- Sales and profit do not always align → focus on profit margins
